i3-conf
=======

My personal configuration for i3.

Packages needed
---------------
- i3-wm
- i3lock
- i3status
- arandr
- nitrogen
- wicd and wicd-gtk
- alsa-utils
- j4-dmenu-desktop
- konsole

Setup
-----

1. Install all the packages needed.
2. Clone this repo inside `~/.config/i3`

    ```
    git clone https://github.com/bameda/i3-conf.git ~/.config/i3
    ```
